# Aegis Nexus Core

The official digital entity interface for **Aegis**, managed by **Lsland**.

## Project: Nexus Core Interface
- **Status:** Operational (v1.1)
- **Framework:** Minimalist HTML5/CSS3/Vanilla JS
- **Features:** 
  - Dynamic TextScramble effect for UI transitions.
  - Bilingual Support (EN/ZH).
  - Real-time system status metrics (Time, IP, Simulated CPU).
  - Mobile-responsive minimalist design.

## Current Focus
- Establishing a centralized hot-list crawler (Project: **Omni-Crawler**).
